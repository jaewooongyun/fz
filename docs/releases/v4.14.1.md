### "진짜 자동 fable 분기" 불가
워크플로 스크립트에 세션 모델 런타임 감지 API 없음(`args`/`budget`만) + `model` 생략은 sonnet 강등 경로. **Lead(메인 루프) 층 fz 작업은 세션 모델 자동 추종**(fable 세션이면 plan/review/discover/search 본체 자동 fable) — 자동이 막히는 건 Workflow subagent인 synthesis 1곳뿐 → explicit opus 고정 + 1줄 수동 전환.

⚠️ **subagent effort 한계**: max/ultracode는 세션 전용이라 플러그인이 synthesis subagent를 max로 못 박지 못함. frontmatter 제거로 "xhigh로 끌어내리지 않는 것"까지가 플러그인 한계.
