### 2. [CRITICAL] Codex Plugin 감지 로직 fix (Cbug-1)

**Codex self-reflexive verify 단독 발견** — Claude family blind spot 정량 실증.

기존 (영원히 false 반환 위험):
```bash
codex mcp list 2>/dev/null | grep -q plugin
```

실측:
```
$ codex mcp list
"No MCP servers configured yet."
```

config.toml에는 `[plugins."github@openai-curated"] enabled = true`가 있고 plugin cache `~/.codex/plugins/cache/openai-curated/github/`이 존재. **MCP server ≠ plugin (계층 다름) → Plugin 모드 영원히 미사용**.

수정 (`fz-codex/SKILL.md` line 103 + 602):
```bash
grep -q '^\[plugins\.' ~/.codex/config.toml 2>/dev/null && \
  ls ~/.codex/plugins/cache/*/ 2>/dev/null | head -1 >/dev/null && \
  PLUGIN_AVAILABLE=true || PLUGIN_AVAILABLE=false
```

→ config.toml + cache 동시 확인. **Plugin 모드 정상 작동 가능**해진 critical fix.
